How to write a SQL query

Overview

SQL stands for Structured Query Language. SQL is a standard language for storing, manipulating and retrieving data in databases. In this tutorial you will learn how to select records from database tables using SQL select statement. In this tutorial, we will use semicolon at the end of each SQL statement.

The SQL SELECT Statement

The SELECT statement is used to select data from a database. The data returned is stored in a result table, called the result-set.

Syntax: 

SELECT column1, column2, ...
FROM table_name;
WHERE condition;

If you want to select all the fields available in the table, use the following syntax:


SELECT * FROM table_name;

Below is an example of a table called “Customers”:

Id Name Gender Age Email
1 John M 21 john@gmail.com
2 Bob M 26 bob@gmail.com
3 Jessica F 22 jessica@gmail.com
4 Zoe F 25 zoe@gamil.com

SELECT * Example

The following SQL statement selects all the columns from the "Customers" table:


SELECT * FROM Customers;

SELECT Column Example

The following SQL statement selects the "Name" column from the "Customers" table:


SELECT Name FROM Customers;

The SQL WHERE Clause

The WHERE clause is used to filter records. It is used to extract only those records that fulfill a specified condition.

Example 1

The following SQL statement selects the customer whose name is John, in the "Customers" table:


SELECT * FROM Customer WHERE Name='John';
Example 2

The following SQL statement selects Names of customers whose are older than 22 in the "Customers" table:


SELECT Name FROM Customer WHERE Age>22;
